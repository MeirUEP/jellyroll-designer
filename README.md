# Jellyroll Battery Cell Designer

A single-page web application for designing and computing tab positions on wound cylindrical battery cells (jellyroll construction).

## Features

- Variable-pitch spiral winding simulation
- 4 interactive views: Side, Top, Unroll, and 3D (Three.js)
- Editable layer stack with drag reordering
- Cathode and anode tab placement computation
- CSV export of tab positions
- JSON save/load of full configuration
- Dark/light theme support

## Usage

Open `index.html` in a browser, or visit the deployed GitHub Pages site.

## Deployment

Automatically deployed to GitHub Pages on push to `main` via GitHub Actions.
